### 简介

`deepcopy-gen` 是一个 Go 代码生成工具，自动扫描指定目录下所有 Go 源文件中的 struct 定义，并为每个 struct 生成类型安全的深拷贝（DeepCopy）方法。生成的代码使用泛型辅助函数，简洁高效，零反射开销。

### 特性

- **AST 精确解析** — 基于 `go/ast` 标准库，准确识别所有 struct 字段类型
- **泛型优化** — 生成的代码调用 `deepcopy` 包的泛型函数，单行完成复杂拷贝
- **全类型覆盖** — 支持基本类型、指针、切片、Map、数组、嵌套结构、interface{}、channel
- **编译期安全** — 生成代码通过 `go/parser` 语法校验，无运行时反射
- **递归子目录** — 自动遍历子目录，按 package 分组输出

### 支持的字段类型

| 类型 | 示例 | 生成代码 |
|---|---|---|
| 基本类型 | `int`, `string`, `bool` | `*out = *in`（值拷贝） |
| 指针 | `*string`, `*int` | `dc.CopyPtr(in.Field)` |
| 双重指针 | `**float64` | `dc.CopyDoublePtr(in.Field)` |
| 切片 | `[]string` | `dc.CopySlice(in.Field)` |
| 指针切片 | `[]*int` | `dc.CopySlicePtr(in.Field)` |
| 嵌套切片 | `[][]float64` | `dc.CopySliceSlice(in.Field)` |
| Map | `map[string]string` | `dc.CopyMap(in.Field)` |
| 指针值 Map | `map[string]*bool` | `dc.CopyMapPtr(in.Field)` |
| 切片值 Map | `map[string][]int` | `dc.CopyMapSlice(in.Field)` |
| 固定数组 | `[3]float64` | `*out = *in`（值拷贝） |
| 指针数组 | `[3]*int` | 逐元素内联拷贝 |
| interface{} | `interface{}` | `*out = *in`（浅拷贝） |

### 生成示例

输入：
```go
package models

type User struct {
    ID       int
    Name     string
    Tags     []string
    Metadata map[string]string
    Manager  *User
}
```

输出（`structinfo.go`）：
```go
// Code generated by deepcopy-gen. DO NOT EDIT.

package models

import (
    dc "deepcopy-gen/deepcopy"
)

func (in *User) DeepCopy() *User {
    if in == nil {
        return nil
    }
    out := new(User)
    *out = *in
    out.Tags = dc.CopySlice(in.Tags)
    out.Metadata = dc.CopyMap(in.Metadata)
    out.Manager = dc.CopyPtr(in.Manager)
    return out
}
```

### 限制

- **数组泛型** — Go 泛型不支持泛型数组长度 `[N]T`，指针数组仍用内联代码
- **循环引用** — 自引用结构（如 `Node.Parent -> Node`）生成浅拷贝指针，不追踪 visited set
- **interface{}** — 仅浅拷贝，无法深拷贝 interface 内部值（需 reflect）
- **channel** — 不做拷贝（channel 不可复制）

### Introduction

`deepcopy-gen` is a Go code generation tool that scans all Go source files in a given directory (recursively) for struct definitions and generates type-safe deep copy (`DeepCopy`) methods. The generated code uses generic helper functions — concise, efficient, and zero reflection overhead.

### Features

- **Precise AST parsing** — Built on `go/ast`, accurately identifies all struct field types
- **Generic optimization** — Generated code calls generic functions from the `deepcopy` package, single-line complex copies
- **Full type coverage** — Supports basic types, pointers, slices, maps, arrays, nested structs, interface{}, channels
- **Compile-time safety** — Generated code validated by `go/parser`, no runtime reflection
- **Recursive scanning** — Automatically traverses subdirectories, groups output by package

### Supported Field Types

| Type | Example | Generated Code |
|---|---|---|
| Basic | `int`, `string`, `bool` | `*out = *in` (value copy) |
| Pointer | `*string`, `*int` | `dc.CopyPtr(in.Field)` |
| Double pointer | `**float64` | `dc.CopyDoublePtr(in.Field)` |
| Slice | `[]string` | `dc.CopySlice(in.Field)` |
| Pointer slice | `[]*int` | `dc.CopySlicePtr(in.Field)` |
| Nested slice | `[][]float64` | `dc.CopySliceSlice(in.Field)` |
| Map | `map[string]string` | `dc.CopyMap(in.Field)` |
| Pointer-value map | `map[string]*bool` | `dc.CopyMapPtr(in.Field)` |
| Slice-value map | `map[string][]int` | `dc.CopyMapSlice(in.Field)` |
| Fixed array | `[3]float64` | `*out = *in` (value copy) |
| Pointer array | `[3]*int` | Inline per-element copy |
| interface{} | `interface{}` | `*out = *in` (shallow copy) |

### Limitations

- **Array generics** — Go generics don't support generic array length `[N]T`; pointer arrays use inline code
- **Circular references** — Self-referential structs (e.g. `Node.Parent -> Node`) generate shallow pointer copies without visited-set tracking
- **interface{}** — Shallow copy only; deep copying interface values requires reflect
- **Channels** — Not copied (channels are not copyable)
